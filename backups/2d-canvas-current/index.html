Physik EF

Raumkrümmung

Didaktisches Modell: Eine Masse krümmt die dargestellte Umgebung. Testkörper und Lichtstrahlen werden qualitativ beeinflusst.

Objekttyp

Leitfragen
  1. Was passiert mit dem Gitter bei größerer Zentralmasse?
  2. Wann entsteht ein Vorbeiflug, wann eine Umlaufbahn?
  3. Wie wird ein naher Lichtstrahl abgelenkt?
  4. Warum ist die Gitter-Eindellung nur ein Modellbild?

Diese Simulation ist ein vereinfachtes, didaktisch überhöhtes Modell. Sie ist keine exakte Rechnung der Allgemeinen Relativitätstheorie: Die Gitter-Eindellung ist eine zweidimensionale Analogie, Testmassen werden Newton-ähnlich berechnet und Lichtablenkung wird qualitativ verstärkt.
